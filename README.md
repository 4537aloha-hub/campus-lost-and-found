# 校园失物招领管理系统

基于 Vue 3 + Node.js 构建的校园失物招领管理系统，提供完整的失物招领功能和管理员后台管理。

## 🎯 项目概述

本系统旨在帮助校园内失物招领信息的高效管理，为失主和拾物者提供便捷的沟通渠道。

## ✨ 亮点功能

### 实时聊天
- 基于 Socket.io 实现实时消息通信
- 支持物品相关的即时沟通
- 消息状态实时同步

### 完整的审核流程
- 失物招领信息提交审核
- 管理员审核确认机制
- 多级审核状态管理

### 图片上传与预览
- 支持轮播图、物品图片上传
- 本地预览功能
- 文件大小和格式校验

### 响应式设计
- 适配多种设备屏幕
- 移动端友好交互

## 🛠️ 技术栈

### 前端
- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Pinia** - Vue 官方状态管理库
- **Element Plus** - Vue 3 组件库
- **Socket.io Client** - 实时通信客户端

### 后端
- **Node.js** - JavaScript 运行时
- **Express** - Web 应用框架
- **MySQL** - 关系型数据库
- **Socket.io** - 实时通信服务端
- **JWT** - 身份认证

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0.0
- MySQL >= 5.7
- npm >= 9.0.0

### 安装与运行

**1. 克隆项目**

```bash
git clone <repository-url>
cd campus-lost-and-found
```

**2. 安装依赖**

```bash
# 安装后端依赖
cd api_server
npm install

# 安装前端依赖
cd ../client
npm install
```

**3. 配置数据库**

创建 MySQL 数据库并配置连接信息：

```bash
# 在 api_server/config.js 中配置数据库连接
```

**4. 启动服务**

```bash
# 启动后端服务 (端口: 3000)
cd api_server
npm run start

# 启动前端服务 (端口: 5173)
cd ../client
npm run dev
```

**5. 访问系统**

- 用户端：http://localhost:5173
- 管理后台：http://localhost:5173/admin

## 🔧 核心功能模块

### 用户端功能

| 模块 | 功能 | 说明 |
|------|------|------|
| 首页 | 轮播图展示、物品推荐 | 展示系统概览 |
